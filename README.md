# Zomato-dashboard
key description
1. Business Problem
Food delivery platforms like Zomato operate across multiple countries, cities, and cuisines.
Key challenges:
Understanding restaurant distribution across regions
Identifying popular cuisines and customer preferences
Tracking ratings vs popularity
Monitoring sales performance and engagement (votes)
Evaluating service features like table booking and online delivery

👉 Without a centralized dashboard, decision-making becomes slow and fragmented.

🎯 2. Purpose of This Dashboard 
To create a single source of truth for restaurant and sales insights
To help stakeholders:
Identify high-performing cities and cuisines
Analyze customer behavior and preferences
Improve business expansion strategies
Optimize restaurant partnerships
Enables data-driven decisions instead of assumptions
